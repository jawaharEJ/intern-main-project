ZEN 1 TECH PARK - 3D Weather Project

Pages Included:
1. Home Page
2. Login Page
3. Locations Page
4. Weather Checking Page
5. Thank You Page

Technologies:
- HTML
- CSS
- JavaScript

Important:
- For accurate weather results, obtain a free API key from OpenWeatherMap and set it in `script.js` at the top: `const OPENWEATHER_API_KEY = "YOUR_API_KEY";`

API Website:
https://openweathermap.org/api

How to Run:
1. Replace `YOUR_API_KEY` in `script.js` with your API key (or leave it to use mock sample data).
2. Open `index.html` in your browser.
3. On the Weather page, enter a city (e.g., Erode, Salem, Coimbatore) and click "Check Weather".
